# House Prices Prediction

Proyecto de Machine Learning orientado a la predicción del precio de viviendas mediante técnicas de análisis de datos y modelos de regresión.

## Dataset

Se ha utilizado el dataset "House Prices - Advanced Regression Techniques" disponible en Kaggle.

El conjunto de datos contiene información sobre características estructurales y económicas de viviendas residenciales.

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Flujo de trabajo

1. Exploratory Data Analysis (EDA)
2. Data Cleaning
3. Feature Engineering
4. Entrenamiento de modelos
5. Evaluación de métricas
6. Visualización de resultados

## Modelos implementados

- Linear Regression
- Random Forest Regressor

## Resultados

| Modelo | MAE | RMSE |
|---|---|---|
| Linear Regression | 0.088 | 0.130 |
| Random Forest | 0.103 | 0.149 |

El modelo de Regresión Lineal obtuvo un mejor rendimiento en este dataset.

## Conclusiones

El proyecto demuestra la importancia del análisis exploratorio y la preparación de datos en problemas de Machine Learning.

Asimismo, se observa que modelos más complejos no siempre ofrecen mejores resultados, dependiendo del tamaño y naturaleza del dataset.
